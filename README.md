# PAI-Project: AI Video Editing SaaS (Backend)

## Project Overview

This repository contains the backend development for the **AI Video Editing SaaS** project. The backend is being developed using **FastAPI (Python)** and provides REST APIs for authentication, project management, video uploads, AI integration, speech processing, and video processing.

## Repository Structure

The project structure is organized around feature modules for different backend components:

* **`development team/`** - Parent directory for feature-specific modules:
  * **`feature-auth-core/`** - Core authentication modules (JWT implementation, user registration, login).
  * **`feature-database-workflow/`** - Database modeling (SQLite), connection setups, and workflows.
  * **`feature-video-processing/`** - FFmpeg integration and video manipulation services.
    * **`feature-ai-speech/`** - Google Gemini and Whisper speech-to-text models.

## Tech Stack

* **Framework:** FastAPI
* **Language:** Python
* **Database:** SQLite
* **Authentication:** JWT
* **AI:** Google Gemini 2.5 Flash
* **Speech-to-Text:** Whisper
* **Video Processing:** FFmpeg
