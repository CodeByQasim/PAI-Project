# AI Video Editing SaaS – Backend

## Project Overview

This repository contains the backend development for the **AI Video Editing SaaS** project. The backend is being developed using **FastAPI (Python)** and provides REST APIs for authentication, project management, video uploads, AI integration, speech processing, and video processing.

## Tech Stack

* **Framework:** FastAPI
* **Language:** Python
* **Database:** SQLite
* **Authentication:** JWT
* **AI:** Google Gemini 2.5 Flash
* **Speech-to-Text:** Whisper
* **Video Processing:** FFmpeg
